# Esercizi C – Lezione 1

Questa cartella contiene gli esercizi della **Lezione 1** del corso di programmazione in C.

## Contenuto

- E01 – Ciao Mondo
- E02 – Ritorno a capo
- E03 – Tabella di colonne
- E04 – ASCII (hex e octal)
- E05 – Range degli interi
- E06 – Range dei numeri reali
- E07 – Overflow con gli interi
- E08 – Underflow con i float
- E09 – Lettura con scanf
- E10 – Operazioni con int
- E11 – Operazioni con double

## Compilazione

Per compilare un esercizio:

```bash
gcc nome_file.c -o programma
```

## Esecuzione

```bash
./programma
```
